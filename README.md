## Installation

1. Download the latest release from the Releases tab.

2. Get a free API key from Groq.

3. Create a `.env` file next to the `.exe` and paste your key:
   ```
   GROQ_API_KEY=your_key_here
   ```

4. Run `GrammarFixer.exe`.
